# 📱 Android Lab 1 – Application Compteur et Message Toast

## 📖 Description
Cette application Android est un projet de travaux pratiques qui démontre des concepts de base du développement Android en utilisant **Java** et **ConstraintLayout**.

L'application contient :
- Un bouton qui affiche un **message Toast** ("Bonjour !")
- Un bouton **compteur** qui incrémente un nombre à chaque clic
- Un **TextView** qui affiche la valeur actuelle du compteur

Ce projet est destiné aux débutants qui apprennent le développement Android.

---

## ⚙️ Technologies utilisées
- Java
- Android Studio
- Android SDK
- ConstraintLayout
- API Toast

---

## 🧩 Fonctionnalités

### 1️⃣ Affichage d’un message Toast
Lorsque l’utilisateur appuie sur le bouton **"Afficher un message"**, un message Toast apparaît à l’écran.

Exemple de message :

```
Bonjour !
```

### 2️⃣ Compteur
Lorsque l’utilisateur appuie sur le bouton **"Compter"** :
- La valeur du compteur augmente de **1**
- Le **TextView** se met à jour automatiquement

---

## 📂 Structure du projet

```
lab1/
│
├── MainActivity.java
├── activity_main.xml
└── README.md
```

### MainActivity.java
Ce fichier gère :
- Les événements de clic des boutons
- La logique du compteur
- L’affichage du message Toast

### activity_main.xml
Ce fichier définit l’interface utilisateur en utilisant **ConstraintLayout** :
- Un TextView pour afficher le compteur
- Un bouton pour afficher le message
- Un bouton pour incrémenter le compteur

---

## ▶️ Comment exécuter le projet

1. Cloner le dépôt

```bash
git clone https://github.com/votre-utilisateur/lab1-android.git
```

2. Ouvrir le projet dans **Android Studio**

3. Connecter un **téléphone Android** ou lancer un **émulateur**

4. Cliquer sur **Run ▶**

---

## 🎯 Objectifs pédagogiques
Ce TP permet de comprendre :
- Le fonctionnement d’une **Activity Android**
- La gestion des événements avec `setOnClickListener`
- La mise à jour dynamique de l’interface utilisateur
- L’utilisation de **Toast**
- L’utilisation de **ConstraintLayout**
